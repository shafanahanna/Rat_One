# Upcline HR Management System

A comprehensive HR management system built with modern web technologies.

## Project Structure

This project consists of two main parts:

- **Backend**: NestJS API with PostgreSQL database
- **Frontend**: React application built with Vite

## Features

- User authentication and authorization
- Employee management
- Department and designation management
- Role-based access control
- Modern, responsive UI
- RESTful API
- File uploads with Cloudinary integration

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- PostgreSQL database

### Backend Setup

```bash
# Navigate to backend directory
cd backend

# Install dependencies
npm install

# Start development server
npm run start:dev
```

For more details, see the [backend README](./backend/README.md).

### Frontend Setup

```bash
# Navigate to frontend directory
cd frontend-vite

# Install dependencies
npm install

# Start development server
npm run dev
```

For more details, see the [frontend README](./frontend-vite/README.md).
